  Portfólio Pessoal – Vitor Madeira
</h1>

Aplicação de portfólio desenvolvida com foco em apresentação profissional, navegação moderna e experiência visual, incluindo seções de introdução, sobre, stacks, projetos e contato com animações suaves.

---

## 📄 Descrição

Este projeto consiste em um portfólio pessoal desenvolvido para praticar e consolidar conceitos fundamentais de front-end moderno, como:

- Componentização com React + TypeScript
- Gerenciamento de estado global com Context API
- Animações com Motion
- Efeitos visuais com tsParticles
- Persistência de tema no Local Storage

A aplicação permite que recrutadores e visitantes conheçam meu perfil, tecnologias dominadas e projetos desenvolvidos, por meio de uma interface responsiva e com identidade visual consistente.

## 🚀 Tecnologias Utilizadas

- React
- TypeScript
- TailwindCSS
- Motion
- Context API
- tsParticles
- React Icons
- Vite

---

## ⚙️ Funcionalidades

- Header fixo com navegação por âncoras
- Alternância de tema claro/escuro
- Persistência do tema no Local Storage
- Hero com destaque profissional e efeito de digitação
- Seção Sobre com apresentação pessoal
- Seção de Stacks dividida por categorias
- Seção de Projetos com cards e links externos
- Seção de Contato com redes e canais diretos
- Fundo animado com partículas na seção inicial
- Menu mobile com navegação responsiva
- Animações de entrada com efeito stagger no scroll
- Layout responsivo para desktop e mobile

## ▶️ Como rodar o projeto localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

```bash
# Clone o repositório
git clone https://github.com/MadeiraVitor/portfolio-vm.git

# Acesse a pasta do projeto
cd portfolio-vm

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O projeto estará disponível em:
http://localhost:5173
